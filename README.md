## Requirements

### Python 3.6

Package requirements are specified in the [requirements.txt](requirements.txt) file.

```
pip3 install -r requirements.txt
```

### Rust

Package requirements are specified in the [Cargo.toml](Cargo.toml) file.

```
cargo build
```
